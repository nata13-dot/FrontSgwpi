```

├───────────────────────────────┤
│ Contenido FULL-WIDTH          │
```

## 🔧 Instalación en Otras Páginas

Para que una página exista use responsividad, simplemente asegúrate de:

1. **Incluir el CSS:**
```html
<link rel="stylesheet" href="/assets/css/app.css">
```

2. **Incluir el JS al final:**
```html
<script src="/assets/js/responsive.js"></script>
```

3. **Usar estructura correcta:**
```html
<div class="content-wrapper">
    <!-- sidebar -->
    <div class="main-content flex-grow-1">
        <!-- contenido -->
    </div>
</div>
```
